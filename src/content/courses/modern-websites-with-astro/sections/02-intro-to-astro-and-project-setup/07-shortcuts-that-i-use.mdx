## Callout Examples

<Callout type="tip">
  Memorize `Cmd + Shift + P` first. It gives you quick access to almost every VS Code command.
</Callout>

<Callout type="info">
  `Cmd + P` is a file switcher. It is one of the fastest ways to jump between files while coding.
</Callout>

<Callout type="warning">
  Shortcut mappings can differ if your keyboard layout or OS-level keybindings are customized.
</Callout>
