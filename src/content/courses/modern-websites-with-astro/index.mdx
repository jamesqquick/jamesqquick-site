## About This Course

A hands-on course to teach you how to build websites using Astro 4.0, the all-in-one framework for the modern web. Completely reworked and updated for Astro 4.0!

With **40+ in-depth lessons** and **5+ hours of video**, you'll build a full-stack Markdown blog from scratch — adding features like tags, pagination, authentication, comments, and more using modern technologies like Tailwind CSS, TypeScript, Xata, and Cloudinary.

## What You'll Learn

- **Static Site Generation vs Server-side Rendering** — multiple strategies Astro provides for rendering pages in the browser
- **Content Collections** — create, organize, and validate your Markdown content using Content Collections with Zod and TypeScript
- **Image Optimization** — use the Astro Image component for optimized images with lazy loading
- **Astro Islands** — integrate interactive UI framework components into your Astro site
- **TypeScript** — define prop types, database table models, and frontmatter schemas
- **API Routes and Forms** — handle form submissions and build API endpoints
- **Authentication** — build a basic auth strategy using Server-side Rendering, Xata (PostgreSQL), and cookies
- **SEO and Meta Tags** — optimize your site for search engines
- **Deploying and Hosting** — get your site live on Vercel and Netlify

## Who Is This For?

This course is built for beginner to intermediate web developers who have a fundamental knowledge of HTML, CSS, and JavaScript and a desire to learn new skills. Framework experience is encouraged but not required.
